An interactive exploratory data analysis (EDA) dashboard built to uncover content production trends, geographical distributions, and historical growth within the Netflix Movies & TV Shows dataset.

> 🔴 **Live Demo:** [Click here to explore the Netflix Dashboard!](https://netflix-data-analysis-me82nckgzzszyz2se762fv.streamlit.app)

## 📖 Overview
This project dives deep into Netflix's catalog to visualize how their content strategy has evolved over time. By applying professional data analysis techniques, this dashboard provides an interactive lens to explore the ratio of movies to TV shows, identify top producing countries, and track content release distributions over the years. 

## ✨ Features
- **Dataset Exploration:** Clean overview of the raw data and statistical summaries.
- **Movies vs. TV Shows Analysis:** Visual comparisons of the platform's content types.
- **Geographical Insights:** Highlighting the top content-producing countries globally.
- **Historical Growth:** Tracking the volume of new content added to the platform over time.
- **Release Year Distribution:** Analyzing the age and release eras of available content.

## 🛠 Tech Stack
- **Language:** Python
- **Data Manipulation:** Pandas
- **Data Visualization:** Matplotlib, Seaborn
- **Web Framework:** Streamlit
