# Netflix Data Analysis Dashboard

This project performs an exploratory data analysis (EDA) on the Netflix Movies & TV Shows dataset to uncover content production trends.

The project includes an interactive dashboard built with Streamlit.

## Features

- Dataset exploration
- Movies vs TV Shows analysis
- Top producing countries
- Content growth over time
- Release year distribution

## Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
